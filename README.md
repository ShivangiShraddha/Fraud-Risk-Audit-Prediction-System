# Fraud-Risk-Audit-Prediction-System
An AI-powered system that performs automated transaction audit, risk assessment, and fraud prediction using machine learning.

** FEATURES **

Fraud probability prediction (predict_proba)

Risk classification (Low / Medium / High)

Fraud percentage calculation

High-risk transaction filtering

Downloadable audit results

Interactive Streamlit dashboard

** ML APPROACH **

Model: Random Forest Classifier

Inspired by Kaggle Credit Card Fraud dataset

Uses engineered enterprise transaction features

Handles class imbalance and risk thresholding

Each transaction is converted into a feature vector and classified using supervised learning.

** TECH STACK **

Python

Scikit-learn

Pandas / NumPy

Streamlit

Joblib
